Docker

Docker är ett verktyg för utvecklare för att kunna köra mjukvara oberoende av vilken operativsystem man jobbar med.

Varför Docker?

Ett vanligt problem för utvecklare är att mjukvara som utvecklas på ett operativsystem, säg Windows, inte går att köra på macOS eller GNU/Linux. En lösning är att skriva ett program för varje operativsystem. Problemet med den lösningen är att det blir mycket extra jobb och programmen kommer troligtvis inte fungera på samma sätt.

Det är detta problemet som Docker löser. Docker gör det möjligt för mjukvara att köra på precis samma sätt oberoende av vilket operativsystem man kör mjukvaran på.

Hur funkar Docker?

Docker skapar ett litet mini-operativsystem, exempelvis Ubuntu, på din dator. Inuti detta operativsystemet körs sedan koden som du har skrivit. På så sätt kan någon som jobbar på en Mac utveckla på samma projekt som någon som jobbar på en Windows-dator.

För att använda docker behöver i huvudsak tre saker:
1. Skapa en fil som heter Dockerfile
2. Bygga en Image med hjälp av Dockerfile
3. Köra vår Image i en Container
