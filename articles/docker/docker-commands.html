Dockerkommandon

Bygga image 

      
        docker image build .
Dockerfile med annat namn 
      
        docker image build --file Dockerfile2 .
Namnge din image 
      
        docker image build --tag projekt1 .

Bygga containrar med docker-compose 

      
        docker-compose build

Starta containrar med docker-compose 

      
        docker-compose up

Stänga av containrar med docker-compose 

      
        docker-compose down

Ta bort oanvända containrar 

      
        docker container prune

Ta bort oanvända images 

      
        docker image prune

Ta bort allt 

      
        docker system prune -a
