Grunder

Clojure är ett funktionellt språk av typen Lisp.

Hello World!

Nedan följer exempel på hur man skriver ett "Hello World!"-program i Clojure: 

      
        ;; Clojure
        (println "Hello World!")

Motsvarande exempel i Python/JavaScript: 

      
        # Python
        print("Hello World!")
      
      
        // JavaScript
        console.log("Hello World!")

Syntax

Clojure-syntax är väldigt enkel på så sätt att all kod skrivs på i princip samma sätt. Koden består i huvudsak av två saker, data/värden och funktioner som på något sätt använder datan. En funktion anropas med startparentes direkt följt av funktionsnamn, sedan alla parametrar separerade med mellanslag och till sist slutparentes. Exempel: 

      
        ;; Clojure
        (funktionsnamn parameter1 parameter2 parameter3)

Motsvarande exempel i Python/JavaScript: 

      
        # Python
        funktionsnamn(parameter1, parameter2, parameter3)
      
      
        // JavaScript
        funktionsnamn(parameter1, parameter2, parameter3);

Hur definieras en funktion?

En funktion definieras med funktionen defn (kort för define function) som tar funktionsnamn som första input, sedan en lista med inputparametrar och efter det all kod som funktionen skall köra. 

      
        ;; Clojure
        (defn funktionsnamn 
         [parameter1 parameter2] 
         (println "Hello World"))

Motsvarande exempel i Python/JavaScript: 

      
        # Python
        def funktionsnamn(parameter1, parameter2):
            print("Hello World!")
      
      
        // JavaScript
        function funktionsnamn(parameter1, parameter2) {
          console.log("Hello World!");
        }

Ett praktiskt exempel

Låt oss skriva en funktion som skriver ut vårt namn och ålder i en hälsningsfras: 

      
        ;; Definiera funktionen
        (defn hälsning
         [namn ålder]
         (println "Hej jag heter" namn "och jag är" ålder "år gammal"))

        ;; Använd funktionen
        (hälsning "Gottfried Wilhelm Leibniz" 374)
